Detailed Description

Clase para guardar datos de contabilidad Esta clase proporciona métodos para insertar registros relacionados con la contabilidad del sistema en la base de datos. Incluye operaciones para guardar recibos, movimientos, documentos, cobranza, CFDIs y otros datos contables.
