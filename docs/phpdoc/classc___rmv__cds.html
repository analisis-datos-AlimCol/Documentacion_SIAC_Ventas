c_Rmv_cds ( )

Constructor de la clase Eliminación CEDIS

Esta clase maneja la eliminación de registros relacionados con CEDIS, incluyendo:

  • Productos en pedidos y órdenes de salida
  • Entradas de productos
  • Ajustes de stock
  • Devoluciones
  • Notificaciones
  • Respaldos de stock
Returns
void
