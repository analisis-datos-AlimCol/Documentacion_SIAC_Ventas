c_Msj_cds ( )

Constructor de la clase Mensajes CEDIS

Esta clase se encarga de mostrar mensajes de diferentes tipos (éxito, atención, alerta, error) con un formato consistente y visualmente atractivo usando Bootstrap y Font Awesome. Los mensajes se muestran en una estructura de tarjeta con íconos y colores distintivos según el tipo.
