◆ conf_detecta_dispositivo()

conf_detecta_dispositivo ( )

Detecta tipo de dispositivo

Detecta si el dispositivo es móvil, tablet o desktop basado en el User-Agent.

Returns
string Clase CSS correspondiente al dispositivo
Exceptions
ExceptionSi no se puede determinar el tipo de dispositivo

Ejemplo de uso: // Detectar dispositivo $dispositivo = $conf->conf_detecta_dispositivo();

Note
Retorna 'mobile' para dispositivos móviles
Retorna 'tablet' para tablets
Retorna 'desktop' por defecto
No requiere parámetros de entrada

Genera código de producto

Genera un código de producto basado en un prefijo y un ID.

Parameters
string$serfPrefijo del código (ej: 'PRO')
string$idvfID del producto
Returns
string Código generado

Ejemplo de uso: // Generar código de producto $codigo = $conf->conf_genera_codigo('PRO', '123'); // Resultado: PRO0124

Note
Formato del código: SER000
Si el ID es 1 dígito: SER00X
Si el ID es 2 dígitos: SER0XX
Si el ID es 3 dígitos: SERXXX

◆ cont_cal_tiempo_x_fechas()

cont_cal_tiempo_x_fechas ( $df)

Calcula tiempo entre fechas

Calcula la diferencia entre dos fechas y retorna un string con el tiempo transcurrido.

Parameters
object$dfObjeto DateTimeInterval con la diferencia entre fechas
Returns
string Tiempo transcurrido en formato legible

Ejemplo de uso: // Calcular tiempo entre fechas $fecha1 = new DateTime('2023-01-01'); $fecha2 = new DateTime('2024-02-03 14:30:45'); $intervalo = $fecha1->diff($fecha2); $tiempo = $conf->cont_cal_tiempo_x_fechas($intervalo);

Note
Retorna string vacío si no hay diferencia
Incluye años, meses, días, horas, minutos y segundos
Usa plural o singular según el valor
Muestra el signo '-' si la fecha es negativa
