cat_cal_tiempo_x_fechas ( $df)

Calcula el tiempo transcurrido entre dos fechas

Esta función calcula el tiempo transcurrido entre una fecha dada y la fecha actual, retornando el resultado en un formato legible que incluye años, meses, días, horas, minutos y segundos según corresponda.

Parameters
DateTime$dfObjeto DateTime que representa la fecha a comparar
Returns
string Texto con el tiempo transcurrido en formato legible

Ejemplo de uso: $fecha = new DateTime("2023-01-01"); $tiempo = cat_cal_tiempo_x_fechas($fecha); // Retorna "1 año 5 meses 3 días"

◆ cat_detecta_dispositivo()

cat_detecta_dispositivo ( )

Detecta el tipo de dispositivo que accede al sistema

Esta función analiza el User-Agent del navegador para determinar si el acceso se realiza desde un dispositivo móvil, tablet o escritorio.

Returns
string Clase CSS que indica el tipo de dispositivo:
  • "desktop" para computadoras de escritorio
  • "tablet" para tablets
  • "mobile" para dispositivos móviles

Ejemplo de uso: $tipo_dispositivo = cat_detecta_dispositivo(); // Retorna "desktop", "tablet" o "mobile" según el dispositivo
