    Ficha Técnica Unificada del Módulo CEDIS - SIAC 2.0 Ventas

    1. Propósito General

    El módulo CEDIS (Centro de Distribución) gestiona el inventario de producto terminado en almacenes centrales, incluyendo:

    • Entradas por producción o migraciones manuales.
    • Salidas por órdenes de salida o traslados.
    • Consultas y ajustes de inventario.
    • Respaldo de datos mediante archivos JSON y generación de reportes PDF.

    Ubicado en la carpeta ventas/, forma parte del sistema SIAC_2_0_Ventas.

    2. Arquitectura General

    Lenguajes y Tecnologías:

    • PHP,
    • JavaScript (jQuery),
    • HTML/CSS, Bootstrap,
    • MySQL,
    • AJAX.

    Patrones: MVC, Singleton (class_wservice.php), Factory (c_View_cds, c_Up_cds).

    Frontend

    • Interfaz basada en formularios (cds_forms.php).
    • Widgets y tablas interactivas (cds_widgets.php, cds_tablas.php).
    • Uso de Select2, iCheck, Datepicker (ver cds_forms.php, líneas 384, 1700, 1777).

    Backend

    • Controlador AJAX: cds_ajax.php.
    • Clase principal: class_cds.php.
    • Conexión a procedimientos almacenados prefijados como sp_cds_*.

    3. Estructura de Archivos Clave

    Archivo Funcionalidad
    cds_ajax.php Puente AJAX entre frontend y backend, validación.
    cds_forms.php Formularios para entradas, salidas, consultas.
    cds_tablas.php Renderización de tablas HTML.
    cds_widgets.php Visualización de KPIs.
    cds_modals.php Ventanas emergentes.
    cds_g_orden_salida.php Gestión de órdenes de salida.
    cds_g_inventarios.php Vista principal de inventario.
    cds_wspace.php Espacio de trabajo del usuario.
    cds_select.php Carga de datos en selects dinámicos.
    view_cds_stk_ent_loc_pt.php Respaldo de stock desde JSON.

    5. Interacción Cliente-Servidor

    JavaScript

    • Archivos: cds_ajax.js, cds_ajax_1.js.
    • Funciones: cds_gcds_os_av_pd_prods_list_view_x_ids(), cds_g_gstk_inicio_graf_view_x_ids().

    AJAX PHP

    • Archivo: cds_ajax.php
    • Validación de tokens (línea 1).
    • Gestión de escaneo y actualizaciones (líneas 66-180).
    • Integración con otros módulos (línea 350+).

    6. Base de Datos

    Procedimientos Almacenados (SPs): sp_cds_*

    Tablas clave:

    • inventario_cedis
    • movimientos_cedis
    • orden_salida
    • prodts_pedido_stock_temp
    • productos

    7. Funcionalidades Clave

    Entradas de producto:

    • Desde escaneo (cds_forms.php) o migración manual (cds_forms_1.php).
    • Flujos definidos en cds_ajax.php (líneas 66-99).

    Salidas:

    • Desde cds_g_orden_salida.php.
    • Validación de stock (class_cds::get_stock_actual()).
    • Reportes PDF (view_cds_av_pd_os.php).

    Consultas de inventario:

    • Métodos: get_movimientos(), respaldo_inventario() en class_cds.php.
    • Visualización: cds_graficas.php, cds_widgets.php.

    8. Seguridad

    • Validación de tokens (cds_ajax.php, línea 1).
    • Control de roles desde módulo global.
    • Sanitización de entradas y uso de POST en formularios.

    9. Integraciones

    • Módulos internos: ventas, contabilidad, compras, RH.
    • Servicios externos: facturación electrónica, impresión, notificaciones.
